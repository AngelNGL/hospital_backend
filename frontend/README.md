# Frontend - Sistema de Citas Medicas

Frontend en React + Vite para consumir el backend de citas medicas.

## Requisitos

- Node.js
- Backend corriendo en `http://127.0.0.1:8010`

## Instalacion

```powershell
cd frontend
npm install
```

## Desarrollo

```powershell
npm run dev
```

URL local:

```text
http://127.0.0.1:3000
```

## Funcionalidades conectadas

- Login con JWT.
- Registro publico de cliente con seleccion de clinica.
- Agendar cita.
- Historial, cancelacion y reprogramacion.
- Vista de agenda para doctor.
- Consumo de catalogos, pacientes, doctores, disponibilidad y citas.
